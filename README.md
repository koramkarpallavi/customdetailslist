# Custom DetailsList (Fluent UI)

A reusable **Custom Filterable + Sortable** Fluent UI `DetailsList` component.

## Features
- Column sorting (text / number / time)
- Column filtering (text, choice, partial choice, number range)
- Optional lazy loading + sticky header
- Optional selection chips (`FilterChip`)

## Repo layout
- `src/` – reusable component + minimal types
- `sample/react-app/` – runnable Vite React + TS sample

## Run the sample
```bash
cd sample/react-app
npm install
npm run dev
```
